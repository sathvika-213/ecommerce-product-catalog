# E-Commerce Product Catalog

## Project Overview
E-Commerce Product Catalog is a full-stack web application that allows users to browse products, search products, add items to cart, buy products, and view order history.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
- Git & GitHub

## Features
- View all products
- Search products by name
- Add products to cart
- Buy products instantly
- Checkout page
- Order history page
- Product images
- Responsive and attractive UI

## Installation

1. Clone the repository

```bash
git clone https://github.com/sathvika-213/ecommerce-product-catalog.git
```

2. Install dependencies

```bash
npm install
```

3. Configure MySQL database and update `.env`

4. Start the server

```bash
npm start
```

5. Open the frontend

```text
http://localhost:5500/frontend/index.html
```
